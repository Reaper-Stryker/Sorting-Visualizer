# 📊 Sorting & Searching Visualizer

[![Live Demo](https://img.shields.io/badge/demo-Live%20View-success)](#) An interactive web application built to help computer science students and developers understand how various sorting and searching algorithms work under the hood. By animating the process step-by-step, this tool visualizes complex logic, variable states, and performance metrics in real-time.

## ✨ Features

* **Interactive Controls:** Media-player style playback controls (Play, Pause, Step Forward, Step Back) to easily track the algorithm's progress.
* **Customizable Data:** Generate random arrays of varying sizes or input your own custom, comma-separated datasets.
* **Dynamic Speed Adjustment:** Control the speed of the visualization on the fly using a smooth range slider.
* **Live Metrics:** Real-time tracking of algorithmic operations, including **Comparisons** and **Swaps**.
* **Color-Coded Visualization:** * 🟦 **Default:** Unsorted elements.
  * 🟨 **Comparing:** Elements currently being evaluated.
  * 🟥 **Swapping:** Elements changing positions.
  * 🟩 **Sorted:** Elements in their final, correct position.

## 🛠️ Tech Stack

This project was built from scratch without the use of heavy frameworks to demonstrate a strong grasp of fundamental web technologies:
* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom styling, flexbox layouts, and UI components (no Bootstrap/Tailwind).
* **Vanilla JavaScript (ES6+):** Algorithm logic, DOM manipulation, and asynchronous animation loops (`async`/`await`).
